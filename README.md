# 自己紹介

このプロジェクトは、私のスキルや経歴を紹介するためのポートフォリオサイトです。

https://it-f-30.github.io/nagashima.shohei/

## 技術スタック

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## 開発環境

- macOS
- Bun

## 開発手順

1. `bun install`で依存関係をインストール
2. `bun run dev`で開発サーバーを起動
3. `bun run build`でビルド
4. `bun run start`で起動
